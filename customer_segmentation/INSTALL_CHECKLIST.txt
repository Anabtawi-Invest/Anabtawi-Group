ODOO 19 INSTALL CHECKLIST - Customer Segmentation v19.0.1.0.1
=============================================================

BEFORE UPLOAD
-------------
[ ] Folder name is exactly: customer_segmentation
[ ] Odoo.sh requirements.txt includes: openpyxl>=3.0.0
[ ] Sales and Purchase apps are installed on the database

UPLOAD TO ODOO.SH
-----------------
Path: /home/odoo/src/user/customer_segmentation/

AFTER BUILD
-----------
[ ] Apps -> Update Apps List
[ ] Install or Upgrade "Customer Segmentation"
[ ] No error popup = success

CONFIGURE USERS
---------------
Settings -> Users -> each staff member:
  Assign ONE team under Sales -> Customer Segmentation:
  - Export Sales Team
  - Local Sales Team
  - Procurement Department
  - POS Team

Admins (Settings access) see all contacts automatically.

USE THE MODULE
--------------
Sales -> Configuration -> Customer Segmentation
  -> Import Customers (Excel)
  -> Import History

Excel row 1 headers: name, mobile (required)
Optional: email, phone, city, country

KNOWN BEHAVIOUR (not bugs)
--------------------------
- Department users only see contacts tagged for their team
- Contacts without a department flag are hidden from department users
- Import sets the correct flag automatically
- Duplicate mobile/email rows are skipped
