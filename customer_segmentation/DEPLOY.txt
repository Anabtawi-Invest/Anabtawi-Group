CUSTOMER SEGMENTATION - ODOO 19 (READY TO DEPLOY)
=================================================

FOLDER TO UPLOAD (this entire folder):
  customer_segmentation

All files are inside. Do not rename the folder.


OPTION A - ODOO.SH (your staging server)
----------------------------------------
1. Upload this "customer_segmentation" folder to your Odoo.sh project:
   Path on server: /home/odoo/src/user/customer_segmentation/

   How:
   - Git: push this folder to your Odoo.sh Git branch, OR
   - Odoo.sh editor: copy files into "user/customer_segmentation/"

2. In Odoo.sh, add to requirements.txt (project root) if not there:
   openpyxl>=3.0.0

3. Wait for the build to finish (Odoo.sh rebuilds automatically).

4. In Odoo: Apps -> Update Apps List -> search "Customer Segmentation" -> Install
   (If already tried before: Upgrade instead of Install)


OPTION B - ZIP FILE
-------------------
Use the file next to this folder:
  customer_segmentation_odoo19.zip

Unzip it. You get one folder: customer_segmentation
Upload that folder to Odoo.sh as in Option A.


AFTER INSTALL
-------------
1. Settings -> Users -> open each user
2. Under Sales, assign ONE team:
   - Export Sales Team
   - Local Sales Team
   - Procurement Department
   - POS Team

3. Import customers:
   Sales -> Configuration -> Customer Segmentation -> Import Customers

4. Excel columns (row 1 headers):
   name | mobile | email | phone | city | country
   (name and mobile are required)
